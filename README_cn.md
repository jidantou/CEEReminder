# 项目工程介绍及特点
本项目基于STC89c516芯片，旨在为广大高考考生提供时钟及倒计时服务。

目前有高考剩余天数计算、当前的时间和日期显示、可自定义时间的2个倒计时等功能。高考日期、当前时间、日期等均可在使用过程中设置。
